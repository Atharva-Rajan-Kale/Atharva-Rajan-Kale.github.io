Projects

SmileScan - Predictive Dental Detection

SmileScan is a comprehensive system designed to address dental issues in underserved rural areas. By combining advanced machine learning models with an intuitive web interface, SmileScan facilitates early detection and diagnosis of common dental problems. This system aims to improve oral health outcomes by providing timely and accurate assessments, thus overcoming barriers like financial constraints and geographical isolation that often hinder access to dental care.

CapSolar

CapSolar analyzes the impact of gases like SOx, COx, NOx, and Benzene on solar panel output using location-specific thresholds. Machine learning algorithms like Naïve Bayes, SVM, and decision trees predict energy fluctuations based on gas concentrations, with data visualized via PowerBI. Arduino sensors collect real-time atmospheric data, uploaded to the cloud for continuous monitoring, providing real-time predictions and automated email alerts for proactive energy management.

NeuroPrint

NeuroPrint leverages DMIT technology and machine learning to analyze fingerprints, extracting minutiae and ridge patterns to assess cognitive traits and brain dominance. It uses a MobileNet V2 CNN model with Python libraries like NumPy, TensorFlow, and OpenCV for image analysis. Results generate personalized career recommendations, with data securely managed through an SQLite database.The solution offers a detailed PDF report that helps individuals understand their innate abilities.

PGApp

PGApp is a user-friendly platform for renting or listing PG accommodations. It allows users to search for PG options or post listings with photos, descriptions, and pricing. Built with React for a dynamic frontend, the app uses Node.js and Prisma for efficient backend data handling and real-time updates. It includes secure authentication, user reviews, and booking management, ensuring a smooth rental process. The integration of modern technologies makes PGApp a robust and scalable solution for managing PG rentals.

Achievements

Implementation of Safety Device for Women

IEEE Publication

Women's safety in India is a pressing issue, with rising crime rates affecting women both at home and in public. To address this, we propose an advanced security system using STM32 (Blue Pill) for the core system, GPS for location tracking, and Node MCU (ESP32) for alerts. A switch button activates the system, ensuring it is cost-effective and user-friendly. This solution is detailed in our IEEE publication.
